=======================================
 A library to access SARAD instruments
=======================================

SARAD is a manufacturer of measuring instruments for Radon, Radon progeny and
radioactivity in aerosols. This library provides an interface to access these
measuring instruments via their proprietary communication protocols via serial
interfaces.

Other pages:

- `SARAD homepage <https://sarad.de>`_
